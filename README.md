# HR Data Generator
Generate HR related data for data analysis, exploration and demoing use cases down stream. 

The dataset has the following tables. 

### Table 1: Employees

| Columns | type | 
|----------|----------|
| `employee_id`  | `int` | 
| `age` | `int`  | 
| `gender` | `string`  |
| `first_name` | `string` |
| `last_name` | `string` |
| `hiring_date` | `datetime` |
| `country` | `string` |
| `employment_type` | `string` | 

### Table 2: Organisation

| Columns | type | 
|----------|----------|
| `organization_id`  | `varchar` | 
| `organization_id` | `string`  | 
| `parent_organization_id` | `varchar`  |
| `cost_center` | `varchar` |
| `business_unit` | `string` |
